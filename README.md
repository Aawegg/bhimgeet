# BhimGeet - भीमगीत ☸️

A beautiful mobile-friendly web app to browse, play, and read lyrics of Bhimrao Ambedkar songs.

## Features

- **25+ Songs** - Curated collection of Bhim Geet, Buddha Vandana, Jayanti songs, and more
- **Full Lyrics** - Read complete lyrics in Hindi/Marathi Devanagari script
- **Play on YouTube** - One-tap to play any song on YouTube
- **Categories** - Browse by Jayanti, Bhim Geet, Buddha Vandana, Dhamma Geet, Vidrohi Geet, Marathi
- **Search** - Search songs by title, artist, or lyrics
- **Favorites** - Save your favorite songs (stored locally)
- **Share** - Share lyrics via WhatsApp, Twitter, or copy to clipboard
- **Adjustable Font Size** - Increase/decrease lyrics font size
- **Works Offline** - PWA with service worker for offline access
- **Installable** - Add to home screen on any phone

## How to Run

### Option 1: Simple Python Server
```bash
cd bhimgeet
python3 -m http.server 8000
```
Then open http://localhost:8000 in your browser.

### Option 2: Simple Node.js Server
```bash
npx serve .
```

### Option 3: Open Directly
Just open `index.html` in any modern web browser.

## Install on Phone

1. Open the app URL in Chrome/Safari on your phone
2. Tap "Add to Home Screen" from the browser menu
3. The app will work like a native app!

## Tech Stack

- Vanilla HTML, CSS, JavaScript (no frameworks)
- Progressive Web App (PWA)
- Google Fonts (Poppins)
- Responsive, mobile-first design
